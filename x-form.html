Як створити username в Telegram


Android:

Відкрий Telegram → Налаштування → Username → придумай унікальне ім'я


iPhone:

Telegram → Settings → Edit profile → Username


Для каналу:

Зайди в канал → Редагувати → Тип каналу → Публічний → створи посилання
