TikTok Shop в Испании: семейный проект

📌 Предложение: Предлагаю объединиться для работы с TikTok Shop в Испании.
Цель: дополнительный доход для пенсионеров + обучение подростков заработку в медиа. Все необходимые ресурсы есть.

1. Как запустить: телефоны, SIM, IP, верификация

  • Телефон: Нужно взять отдельный телефон (отправлю в Испанию чистый Xiaomi (сброс до заводских настроек).
  • SIM-карта: купить prepago любого испанского оператора (Movistar, Vodafone, Orange) — около 10–20€.
  • Регистрация TikTok: выполняет совершеннолетний (бабушка или дедушка). Использовать номер испанской SIM, не входить через Google/Apple ID, связанные с Украиной.
  • Локация и IP: телефон всегда находится в Испании, публикации только оттуда. VPN для обхода геолокации не используются — это небезопасно.
  • Верификация для Affiliate: после набора 1000–2000 подписчиков подается заявка в TikTok Studio → Monetization → TikTok Shop for Creator. Потребуется паспорт совершеннолетнего (украинский подходит, проверяют личность).
  • Документы бизнеса не требуются — работаем как Affiliate Creator.

2. TikTok Shop в Испании: комиссии для креатора

💰 Важно: комиссия для Affiliate Creator (креатора) составляет 5–20% от стоимости заказа.
3.85% — это комиссия TikTok с продавца, нас это не касается. Мы работаем как аффилиаты.
Продавец сам устанавливает процент за продвижение. Средние ставки по категориям:

📱 Электроника, гаджеты

Комиссия: 5–10%
Пример: наушники 50€ → комиссия 2.50–5.00€

👗 Мода, красота, косметика

Комиссия: 10–15%
Пример: косметика 40€ → комиссия 4.00–6.00€

🏠 Товары для дома, кухня

Комиссия: 8–12%
Пример: органайзер 25€ → комиссия 2.00–3.00€

⚽ Спорт, фитнес, активный отдых

Комиссия: 12–20%
Пример: фитнес-резинки 30€ → комиссия 3.60–6.00€

🚗 Автотовары, инструмент

Комиссия: 8–15%
Пример: автоаксессуар 45€ → комиссия 3.60–6.75€

🍎 Здоровье, Wellness

Комиссия: 15–20%
Самые высокие ставки, премиум-сегмент
📊 Реальный пример заработка:
• Вы прорекламировали товар для дома (комиссия 12%). Цена 100€.
• 50 продаж в месяц → 100€ × 12% = 12€ с одной продажи.
• Итого: 600€/мес. только с одного товара.
• При 5 активных товарах (разные ниши) → 2000–3000€/мес.

🔥 Рекорды в Испании (2026):
• Креаторы с 20–50 тыс. подписчиков зарабатывают 1500–4000€/мес.
• Топ-аффилиаты с виральными видео (1+ млн просмотров) выходят на 8000–15000€/мес.
• Украинская диаспора в Испании (200 000+) — наша уникальная аудитория с высоким доверием.
