Предложение: объединиться для работы с TikTok Shop в Испании. Цель — дополнительный доход для пенсионеров и обучение подростков заработку в медиа. Необходимые ресурсы можно собрать поэтапно.

  • Телефон: отдельный аппарат (например, сброшенный до заводских настроек), который физически находится в Испании.
  • SIM: prepaid испанского оператора (Movistar, Vodafone, Orange) — ориентир 10–20 €.
  • Регистрация TikTok: совершеннолетний; номер испанской SIM; не входить через Google/Apple ID, привязанные к Украине.
  • Локация: публикации только из Испании; VPN для обхода геолокации не рекомендуется.
  • Affiliate: после 1000–2000 подписчиков — заявка в TikTok Studio → Monetization → TikTok Shop for Creator; паспорт совершеннолетнего для проверки личности.
  • Бизнес-документы для старта как affiliate creator не обязательны.

Для affiliate creator комиссия часто 5–20% с заказа (ставку задаёт продавец). Процент TikTok с продавца (например, ~3,85%) креатора напрямую не касается.

Типы подачи: испанский закадровый голос; украинский или русский в кадре/за кадром; без слов (демонстрация + музыка + текст на экране).

Производство: съёмка и монтаж в одной стране → готовые ролики семье в Испании для публикации с испанского телефона и аффилиатной ссылкой.

Важно про рынок Испании: TikTok Shop здесь — относительно новый формат продаж. Поэтому если взять уже успешные американские видео (с большим числом просмотров, комментариев и сильной статистикой) и озвучить их по-испански (или адаптировать под локальный тон), результат можно получить быстрее и проще, чем строить всё с нуля без проверенных референсов.
  • Короткий вертикальный формат, понятный показ товара, текст на экране, призыв к действию.
  • Длительность шоп-роликов часто 15–60 с; сильный хук в первые 1–3 с.

Как правильно набирать первых подписчиков. Первые 1000–2000 подписчиков должны быть исключительно из Испании.

Как это сделать:

  • Подписывайтесь на испанских блогеров, комментируйте их ролики по-испански — так чаще приходит внимание испанской аудитории.
  • Публикуйте первые видео на испанском или с испанскими субтитрами.
  • Используйте локальные хештеги: #tiktokespaña #parati #españa #tiktokshopspain
  • Не распространяйте ссылку на аккаунт в украинских чатах и не просите украинских друзей подписываться на старте.

Вывод: украинские подписчики из Украины на первом этапе вредны для гео и рекомендаций. Их можно подключать позже, когда уже есть стабильная испанская база.

Фаза 1 (первые 2–3 недели): набор испанской аудитории. Контент на испанском (voiceover может делать член семьи с сильным испанским). Темы: например автотовары или обзоры популярных гаджетов. Цель — быстро набрать 2000+ испанских подписчиков.

Фаза 2 (после доступа к Affiliate Center): добавить украинский контент для украинцев в Испании; задействовать разных членов семьи — кулинария, авто, спорт. Параллельно продолжать испанский контент для масштабирования.

Фаза 3 (масштабирование): несколько аккаунтов под разные ниши (отдельно авто, отдельно кухня и т.д.); использовать TikTok Shop Affiliate Center для товаров с высокой комиссией.

Часто подписчики приходят с простых роликов: юмор, тёплые бытовые сцены, рыбалка, кухня, дед и внук — ищите идеи по хештегам вроде «юмор», «семья», «кухня», «рыбалка», «бабушка и внук».

  1. Короткий юмор или абсурд — иногда только текст на экране, без озвучки.
  2. Дедушка + внук — тёплая сцена, помощь, быт (хорошо заходит локальной аудитории).
  3. Рыбалка / гараж / мастерская — процесс крупным планом, минимум слов.
  4. Кухня / рецепт — ASMR, звук, результат блюда.
  5. Распаковка гаджета с испанской озвучкой по мотивам трендовых US-роликов.
  6. Челлендж 15–30 с с #parati и локальными тегами.
  7. Лайфхак «до/после» — один предмет, понятный эффект, испанский текст на экране.

Схема: монтаж → отправка файла семье → публикация с испанского телефона + аффилиатная ссылка на товар из TikTok Shop España.

Почему суммы ориентировочные? Реальный RPM зависит от ниши, сезона и качества роликов.

Откуда средняя длительность видео? В коде заданы константы AVG_SEC_PER_VIDEO и AVG_VIEWS_PER_VIDEO — подставьте свои средние из статистики.

Модель дохода в калькуляторе упрощена: используются условные € за 1000 просмотров по категориям. Подстройте коэффициенты в скрипте под свои данные из TikTok Shop.
