Архітектура CRM BEROO v1.0

Технічне завдання та аналіз бізнес-логіки для розробника

1. Профіль Компанії (Company Data)

Функціонал: Управління назвою, описом (200 симв.), контактами (Viber/TG), соцмережами та банківськими реквізитами (IBAN/ЄДРПОУ).

Аналіз та порівняння:

Переваги (Strong Points) Слабкі місця vs Global (Weak Points)
+ Глибока інтеграція з TikTok та Bigo Live (нішова спеціалізація на Live-продажах). - Відсутність підтримки кількох юридичних осіб або ФОПів в одному кабінеті (як на Amazon Seller Central).
+ Чітке розмежування між брендом (магазином) та юридичною назвою (для оферти). - Відсутність автоматичної перевірки IBAN на валідність при введенні.

2. Telegram Бот & Автоматизація

Логіка: Керування токеном, статусом роботи бота та текстом привітання (/start).

Західний досвід (TikTok Shop / Amazon):

Amazon/Temu: Використовують AI-чатботів для миттєвого вирішення диспутів.
BEROO: Зараз бот — це вітрина.
Порада: Додати логіку "Abandoned Cart" (Покинутий кошик). Якщо клієнт натиснув "Купити" в боті, але не оплатив WayForPay — бот має нагадати через 2 години.

3. Модуль Доставки (Нова Пошта)

Функціонал: API інтеграція, вибір відділень/поштоматів, автозаповнення ваги та опису.
BEROO (Current) Global Standards (AliExpress/Rozetka)
Ручне керування відділенням відправника. Multi-warehouse: Система сама обирає склад, де є товар у наявності.
Фіксована вага за замовчуванням. Smart Calculation: Вага рахується автоматично на основі суми ваги товарів у кошику.

4. Фінансовий шлюз (WayForPay / Monobank)

Логіка: Прийом платежів через Merchant ID та Secret Key. Автоматична зміна статусу замовлення після успішного Callback від платіжної системи.

Критичний мінус: Немає логіки "Split Payment" (як на Kaspi або іноземних BNPL сервісах — Buy Now Pay Later).
Плюс: Пряма інтеграція з українськими банками, що мінімізує комісію за конвертацію.

5. Маркетинг (Google Merchant / SEO)

Функціонал: Генерація XML-фіду, GTM, GA4, Meta-теги підтвердження власності сайту.

Стратегічний розвиток:

  • Amazon Style: Впровадження системи внутрішніх відгуків, які впливають на рейтинг товару у фіді.
  • TikTok Shop Style: Інтеграція пікселя TikTok для відстеження конверсій саме з прямих ефірів.
